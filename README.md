# flics-2025

This repository contains implementations of the experiments from the paper:

> Dmitrij Schlesinger and Boris Flach, _"Game-theoretic distributed learning of generative models for heterogeneous data collections"_, published at FLICS 2025 (the official link follows).

For experiments from different sections, see the corresponding README-s:
1. IV-A: [MNIST](./mnist/README.md)
2. IV-B: [FMNIST](./fmnist/README.md)
3. IV-C: [PolyMNIST](./polymnist/README.md)
